# Code of Conduct

## Our Standard

We want this project to be welcoming, practical, and respectful.

Examples of positive behavior include:

- being respectful and constructive
- discussing problems in good faith
- giving actionable feedback
- staying focused on the project and its users

Examples of unacceptable behavior include:

- harassment or personal attacks
- insulting, hostile, or exclusionary language
- bad-faith trolling or deliberate disruption
- sharing private or sensitive information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull requests, or other contributions that violate this Code of Conduct.

## Scope

This Code of Conduct applies in project spaces such as issues, pull requests, discussions, and review threads.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the maintainer through a private contact channel where possible.
